# 프랙탈 위상 기반 양자 회로 연구 (서전7년)

이 저장소는 양자 컴퓨터의 '바렌 고원' 문제를 해결하기 위한 **프랙탈 안사츠** 연구 코드를 담고 있습니다.

## 핵심 내용
- **문제점:** 큐비트가 많아지면 신호가 묽어지는 현상 발생 (바렌 고원)
- **해결책:** `i & j == 0` 규칙을 이용한 프랙탈 통로 설계
- **결과:** 기존 방식 대비 신호 보존 능력이 비약적으로 향상됨 (N=200 통계 검증 완료)

## 파일 설명
- `main.py`: 3개 그룹 비교 및 통계 검정 코드
- `final_analysis.png`: 큐비트 숫자에 따른 성능 비교 그래프
